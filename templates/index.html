Project Introduction

Welcome to the home page of my Data Rep Project. The brief for the project is to upload a Single Page Application (SPA )file to github, connect to a temporary server using pouchdb, and and use a combination of Ajax and JSON to store and retrieve the data. The goal is to be able to access data on other pages etc without needing to reload the webpage.The list in the next panel is a compilation of the various programming languages implemented in the project to connect to the server, build the page, and asynchronously access data.
